    AgentArray

    AgentArray

    Subclass of Array with convenience methods used by NetLogo. Tipically the items in the array are Objects but can be any type.

    Constructor

    # new AgentArray(…args)

    Creates an instance of AgentArray. Simply pass-through to super() now, but may add initialization code later.

    Parameters:
    Name Type Attributes Description
    args * <repeatable>

    Zero or more items in Array
    Source:
    • AgentArray.js, line 7
    Example 
    let aa = new AgentArray({x:0,y:0}, {x:0,y:1}, {x:1,y:0})
 //=>  [{ x: 0, y: 0 }, { x: 0, y: 1 }, { x: 1, y: 0 }]
